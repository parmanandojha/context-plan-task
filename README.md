# Study Session Planner

React app using **useState**, **React Hook Form**, **Context API**, and **Tailwind CSS**.

## Scripts

- `npm install` — install dependencies
- `npm run dev` — start dev server
- `npm run build` — production build

## Structure

- `src/context/SessionContext.jsx` — global `sessions`, `addSession`, `deleteSession`, and bonus `totalDurationMinutes`
- `src/components/SessionForm.jsx` — form with React Hook Form
- `src/components/SessionList.jsx` — list + total duration
- `src/components/SessionCard.jsx` — card with priority colors and delete
